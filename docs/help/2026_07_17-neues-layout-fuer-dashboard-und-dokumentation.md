# Neues Layout für Dashboard und Dokumentation

## Datum

2026-07-17

## Bereich

Dashboard, Navigation und Dokumentation

## Kurzbeschreibung

Das Erscheinungsbild der Anwendung wurde grundlegend vereinfacht und vereinheitlicht. Dashboard und Dokumentation verwenden nun dieselbe klare Designsprache mit mehr Weißraum, ruhiger Typografie und einer übersichtlicheren Navigation.

## Was ist neu?

- Ein reduziertes, editoriales Layout ersetzt die bisherige kartenlastige Darstellung.
- Das Dashboard zeigt Konto, Abonnement, Sicherheitscode und Download in einem klar gegliederten Raster.
- Die Navigation ist auf großen Bildschirmen dauerhaft sichtbar und auf Mobilgeräten als gut erreichbare Leiste am unteren Bildschirmrand verfügbar.
- Die Dokumentation besitzt eine ruhigere Lesespalte, klarere Überschriften und eine vereinfachte Seiten- und Inhaltsnavigation.
- Farben, Abstände, Rundungen und Typografie sind über alle Bereiche hinweg konsistent.

## Warum wurde das geändert?

Die bisherige Oberfläche wirkte visuell unruhig und erschwerte das schnelle Erfassen wichtiger Informationen. Das neue Layout stellt Inhalte und Bedienbarkeit in den Vordergrund und schafft eine verlässliche Grundlage für weitere App-Funktionen.

## Wie funktioniert es?

1. Nach der Anmeldung öffnet sich das Dashboard mit einer kompakten Übersicht aller wichtigen Kontoinformationen.
2. Auf dem Desktop befinden sich die Hauptbereiche links in einer festen Navigation.
3. Auf kleinen Bildschirmen wechselt die Navigation automatisch an den unteren Bildschirmrand.
4. In der Dokumentation bleibt die Artikelnavigation auf großen Bildschirmen sichtbar; mobil wird sie über das Menü geöffnet.
5. Die Darstellung passt Schriftgrößen, Abstände und Spalten automatisch an die verfügbare Bildschirmbreite an.

## Betroffene Bereiche

- Dashboard
- Hauptnavigation
- App-Kopfzeile
- Download-Bereich
- Dokumentationsnavigation
- Dokumentationsartikel
- Helles und dunkles Farbschema

## Wichtige Hinweise

Alle bestehenden Funktionen und Inhalte bleiben erhalten. Die Änderung betrifft die Darstellung, Anordnung und responsive Bedienung.

## Beispiel

Auf einem Smartphone erscheinen Konto- und Abonnementstatus untereinander. Die Navigation bleibt am unteren Bildschirmrand erreichbar, ohne den Inhalt zu verdecken.

## Technische Notizen

Das Layout basiert weiterhin auf Inertia, Vue und Tailwind CSS. Die bestehenden Farbmodus-Einstellungen werden weiterhin unterstützt.
